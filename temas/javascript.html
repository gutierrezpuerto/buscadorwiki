¿Qué es JavaScript?

JavaScriptes un lenguaje de programación que permite agregar interactividad a las páginas web. Se ejecuta en el navegador y es uno de los lenguajes fundamentales del desarrollo web junto a HTML y CSS.


¿Para qué se usa?

  • Actualizar el contenido de una página sin recargarla.
  • Validar formularios.
  • Crear efectos visuales (animaciones, sliders).
  • Manipular el DOM (estructura del documento HTML).
  • Hacer peticiones a servidores (AJAX, fetch).
